# 🚀 EC2 Cost Optimization

## 🔍 Problem

Many EC2 instances run without proper utilization, leading to high costs.

---

## ✅ Solutions

### 1. Stop Idle Instances

* Identify using CloudWatch metrics
* Stop instances during non-working hours

### 2. Rightsize Instances

* Move from large instances to smaller ones
* Example:

  * m5.large → t3.medium → save ~40%

### 3. Use Reserved Instances

* Save up to 60% for long-term workloads

### 4. Use Spot Instances

* Ideal for non-critical workloads
